# Variables

[http://docs.ansible.com/ansible/playbooks_variables.html]

Example contents:
- [demo playbook](vars_example.yml)
- [sample inventory](inventory)
- [host variables file](host_vars.yml)

Example of variable scopes, accessing host variables via hostvars
```sh
ansible-playbook vars_example.yml -i inventory
```
```sh
PLAY [localhost] **************************************************************************************************************************************************************************************************

TASK [Gathering Facts] ********************************************************************************************************************************************************************************************
ok: [localhost]

TASK [debug] ******************************************************************************************************************************************************************************************************
ok: [localhost] => {
    "myvar": "myvar defined in group vars"
}

PLAY [localhost] **************************************************************************************************************************************************************************************************

TASK [Gathering Facts] ********************************************************************************************************************************************************************************************
ok: [localhost]

TASK [debug] ******************************************************************************************************************************************************************************************************
ok: [localhost] => {
    "myvar": "myvar defined in playbook"
}

PLAY [localhost] **************************************************************************************************************************************************************************************************

TASK [Gathering Facts] ********************************************************************************************************************************************************************************************
ok: [localhost]

TASK [Variables from Play scope] **********************************************************************************************************************************************************************************
ok: [localhost] => (item=myvar2) => {
    "item": "myvar2", 
    "myvar2": "myvar2 defined in vars section"
}
ok: [localhost] => (item=myvar) => {
    "item": "myvar", 
    "myvar": "myvar defined in vars section"
}

TASK [Variables from Host scope] **********************************************************************************************************************************************************************************
ok: [localhost]

TASK [debug] ******************************************************************************************************************************************************************************************************
ok: [localhost] => (item=myvar) => {
    "hostvars[inventory_hostname][\"myvar\"]": "myvar defined in the host scope", 
    "item": "myvar"
}
ok: [localhost] => (item=myvar2) => {
    "hostvars[inventory_hostname][\"myvar2\"]": "myvar2 defined in the host scope", 
    "item": "myvar2"
}

PLAY [localhost] **************************************************************************************************************************************************************************************************

TASK [Gathering Facts] ********************************************************************************************************************************************************************************************
ok: [localhost]

TASK [Example of the builtin variable ansible-version] ************************************************************************************************************************************************************
ok: [localhost] => {
    "ansible_version": {
        "full": "2.3.1.0", 
        "major": 2, 
        "minor": 3, 
        "revision": 1, 
        "string": "2.3.1.0"
    }
}

PLAY RECAP ********************************************************************************************************************************************************************************************************
localhost                  : ok=10   changed=0    unreachable=0    failed=0   

```

Example of passing variables on the command line
```sh
$ ansible-playbook vars_example.yml -i inventory -e "myvar='myvar defined in the command line'"
```
```sh
PLAY ***************************************************************************

TASK [setup] *******************************************************************
ok: [localhost]

TASK [debug] *******************************************************************
ok: [localhost] => {
    "myvar": "myvar defined in the command line"
}

PLAY ***************************************************************************

TASK [setup] *******************************************************************
ok: [localhost]

TASK [debug] *******************************************************************
ok: [localhost] => {
    "myvar": "myvar defined in the command line"
}

PLAY ***************************************************************************

TASK [setup] *******************************************************************
ok: [localhost]

TASK [Example of the builtin variable ansible-version] *************************
ok: [localhost] => {
    "ansible_version": {
        "full": "2.0.0.2", 
        "major": 2, 
        "minor": 0, 
        "revision": 0, 
        "string": "2.0.0.2"
    }
}

PLAY RECAP *********************************************************************
localhost                  : ok=6    changed=0    unreachable=0    failed=0   

```
